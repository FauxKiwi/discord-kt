auditLogs

suspend fun auditLogs(limit: Int = 100, before: DateTime? = null, after: DateTime? = null, oldestFirst: Boolean = after != null, user: Snowflake, action: AuditLogAction): Sequence<AuditLogEntry>
Content copied to clipboard

Returns a sequence that enables receiving the guild’s audit logs.

You must have the view_audit_log permission to use this.

Return

The audit log entries

Parameters

limit

The number of entries to retrieve. If -1 retrieve all entries.

before

Retrieve entries before this date.

after

Retrieve entries after this date.

oldestFirst

If set to true, return entries in oldest->newest order. Defaults to true if after is specified, otherwise false

user

The moderator to filter entries from.

action

The action to filter with.

Throws

discord.exceptions.Forbidden

You are not allowed to fetch audit logs

discord.exceptions.HTTPException

An error occurred while fetching the audit logs.
